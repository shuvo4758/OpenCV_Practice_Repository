# OpenCV_Practice_Repository

Overview
This repository contains my practice work and learning materials for OpenCV. It includes Jupyter Notebook files and Python scripts where I implement fundamental computer vision techniques. The main goal of this repository is to build practical skills in image processing and real-time video analysis using OpenCV.

--Repository Contents--
Image processing basics (reading, writing, resizing, cropping images)
Drawing shapes and adding text on images
Image transformations (grayscale, blur, edge detection)
Contour detection and object tracking
Face detection using Haar Cascade
Real-time video processing using webcam
